API Biblioteki Uniwersyteckiej

Dostępne Endpointy API

GET /api/Ksiazki
Pobierz listę wszystkich książek
GET /api/Ksiazki/{id}
Pobierz szczegóły książki o podanym ID
GET /api/Ksiazki/search?query={zapytanie}
Wyszukaj książki zawierające podaną frazę w tytule, autorze lub kategorii
POST /api/Ksiazki
Dodaj nową książkę
PUT /api/Ksiazki/{id}
Aktualizuj istniejącą książkę
DELETE /api/Ksiazki/{id}
Usuń książkę
GET /api/Kategorie
Pobierz wszystkie kategorie książek
GET /api/Wypozyczenia
Pobierz wszystkie wypożyczenia
POST /api/Wypozyczenia/{id}/zwrot
Oznacz książkę jako zwróconą
GET /api/Search?query={zapytanie}
Wyszukiwanie proste
GET /api/Search/advanced?title={tytul}&author={autor}&categoryId={idKategorii}&yearFrom={rokOd}&yearTo={rokDo}
Wyszukiwanie zaawansowane
