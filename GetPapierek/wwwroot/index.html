University Library API

Available API Endpoints

GET /api/Books
Get list of all books
GET /api/Books/{id}
Get details of a book by ID
GET /api/Books/search?query={query}
Search for books containing the provided phrase in title, author or category
POST /api/Books
Add a new book
PUT /api/Books/{id}
Update an existing book
DELETE /api/Books/{id}
Delete a book
GET /api/Category
Get all book categories
GET /api/Rental
Get all rentals
POST /api/Rental/{id}/return
Mark book as returned
GET /api/Search?query={query}
Simple search
GET /api/Search/advanced?title={title}&author={author}&categoryId={categoryId}&yearFrom={yearFrom}&yearTo={yearTo}
Advanced search
